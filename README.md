# MedBuddy ML Capstone

A full-stack AI-powered healthcare assistant application built using **Streamlit**, **FastAPI**, and **Machine Learning models**.
This project is designed to provide intelligent health-related predictions and insights through an interactive web interface and scalable backend API.

---

## 🚀 Project Overview

**MedBuddy ML Capstone** is an end-to-end machine learning application that combines:

* **Frontend:** Streamlit
* **Backend API:** FastAPI
* **Machine Learning Model:** Trained ML model for prediction/analysis
* **Deployment Ready Architecture**

The application allows users to input medical or health-related data and receive predictions generated by the trained machine learning model.

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/medbuddy_ml_capestone.git
cd medbuddy_ml_capestone
```

---

## 2️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Running the Application

## Start FastAPI Backend

```bash
cd backend
uvicorn main:app --reload
```

Backend runs on:

```bash
http://127.0.0.1:8000
```

API Docs:

```bash
http://127.0.0.1:8000/docs
```

---

## Start Streamlit Frontend

Open a new terminal and run:

```bash
cd frontend
streamlit run app.py
```

Frontend runs on:

```bash
http://localhost:8501
```
